最小二乘拟合的理解

  最小二乘法一种数学回归分析的一种方法，通常用于数据拟合，在误差估计、不确定度、系统辨识及预测、预报等数据处理诸多学科领域得到广泛应用。起源于十八世纪的大航海探索时期，发展于天文领域和航海领域。法国科学家勒让德于1805年首先提出了最小二乘法，后来在1822年由高斯证明了最小二乘法的优越性，也就是高斯-马尔可夫定理。其中最小二乘其实是日语的翻译，二乘在日语中也就是平方的翻译，中文其实可以称为最小平方法。

一、基本定义及特性

  最小二乘法（又称最小平方法）通过最小化误差的平方和寻找数据的最佳函数匹配。利用最小二乘法可以简便地求得未知的数据，并使得这些求得的数据与实际数据之间误差的平方和为最小。最小二乘法还可用于曲线拟合，其他一些优化问题也可通过最小化能量或最大化熵用最小二乘法来表达。

(1)问题定义

若存在一对观测量$ ( { {},y} ) \(，其中\){} = {^T} \(，\)y R$，且满足以下函数： $$ y = f\left( {{\bf{x}},{\bf{w}}} \right) $$ 其中\({\bf{w} } = {\left[ { {w_1},{w_2},{w_3}, \cdots ,{w_n} } \right]^T}\)为待定参数。

为了寻找函数\(y = f\left( { {\bf{x} },{\bf{w} } } \right)\)的最佳估计值，对于给定的\(m\)组（通常\(m>n\)）的观测数据\(\left( { { {\bf{x} }^{\left( i \right)} },{y^{\left( i \right)} } } \right)\)，其中\(i = 1,2,3, \cdots ,m\)，求解目标函数： $$ L\left( {y,f\left( {{\bf{x}},{\bf{w}}} \right)} \right) = {\sum\limits_{i = 1}^m {\left[ {{y^{\left( i \right)}} - f\left( {{{\bf{x}}^{\left( i \right)}},{\bf{w}}} \right)} \right]} ^2} \label{eq:1} \tag{1} $$ 取最小值时的参数\({ {\bf{w} }^*} = {\left[ {w_1^*,w_2^*,w_3^*, \cdots ,w_n^*} \right]^T}\)。求解这类问题称为最小二乘问题，也被称为最小二乘拟合。当\(y = f\left( { {\bf{x} },{\bf{w} } } \right)\)为线性函数时，则该问题是线性最小二乘，否则称为非线性最小二乘。

2.缺点

二、线性最小二乘问题

  当拟合函数\(y = f\left( { {\bf{x} },{\bf{w} } } \right)\)为线性函数时，则称求解上式\(\eqref{eq:1}\)目标函数的问题为线性最小二乘问题。

  例如当给定\(m\)组观测值\(\left( { {x^{\left( i \right)}},{y^{\left( i \right)} } } \right)\)其中\(i = 1,2,3, \cdots ,m\)，求拟合方程\(y=ax+b\)中的最佳参数\(a\)和\(b\)使得目标函数\(\sum\limits_{i = 1}^m { { {\left[ { {y^{\left( i \right)} } - \left( {a{x^{\left( i \right)} } + b} \right)} \right]}^2}}\)最小。

将多个观测值带入可获得多个等式方程： \[ \begin{array}{l} {y^{\left( 1 \right)}} = a{x^{\left( 1 \right)}} + b\\ {y^{\left( 2 \right)}} = a{x^{\left( 2 \right)}} + b\\ {y^{\left( 3 \right)}} = a{x^{\left( 3 \right)}} + b\\ \cdots \\ {y^{\left( m \right)}} = a{x^{\left( m \right)}} + b \end{array} $$ 求解参数为${\bf{w} } = \left[ { {w_1},{w_2} } \right]$即${w_1}{\rm{ = } }a,{w_2} = b$，求解参数为2个，则观测值至少要有3组，否则不存在优化问题，因为该参数已经可以唯一确定了。所以考虑当给定3组观测值时，则有： \] \[\begin{array}{l} {y^{\left( 1 \right)}} = w_1{x^{\left( 1 \right)}} + b\\ {y^{\left( 2 \right)}} = w_1{x^{\left( 2 \right)}} + b\\ {y^{\left( 3 \right)}} = w_1{x^{\left( 3 \right)}} + b \end{array}\]

\[ 写成向量形式： \] = {w_1}+ {w_2}= = {} \[ 则可以理解为1个三维向量由两外2个三维的基向量组成，两个基向量分别是${v_1} = {\left[ { {x^{\left( 1 \right)} },{x^{\left( 2 \right)} },{x^{\left( 3 \right)} } } \right]^T}$和${v_2} = {\left[ {1,1,1} \right]^T}$，若在三维空间里（对应3组观测值），由于$a$和$b$可以取任意值，所以这两个基向量可以理解为形成了一个空间平面，而当${\left[ { {y^{\left( 1 \right)} },{y^{\left( 2 \right)} },{y^{\left( 3 \right)} } } \right]^T}$不在此平面内，则我们无法求得与它重合的向量，那么我们需要寻求一个最接近它的点（距离最近），而一个点与一个平面最近的点就是这个点与这个平面的垂线相交点，交点为${\bf{A} }{ {\bf{w} }^*}$且有向量${\bf{y} } - {\bf{A} }{ {\bf{w} }^*}$与平面${\bf{Aw} }$垂直则有： \] {\left( {{\bf{Aw}}} \right)^T}\left( {{\bf{y}} - {\bf{A}}{{\bf{w}}^*}} \right) = 0 \label{eq:2} \tag{2} $$ 原理示意如下图所示：

当观测数据变多时，则基向量由原来三维变为高维向量，但其垂直关系不变。仍以三维空间为例，由上式\(\eqref{eq:2}\)进一步化简： \[ \begin{array}{l} {\left( {{\bf{Aw}}} \right)^T}\left( {{\bf{y}} - {\bf{A}}{{\bf{w}}^*}} \right) = 0\\ {{\bf{w}}^T}{{\bf{A}}^T}\left( {{\bf{y}} - {\bf{A}}{{\bf{w}}^*}} \right) = 0\\ {{\bf{w}}^T}\left( {{{\bf{A}}^T}{\bf{y}} - {{\bf{A}}^T}{\bf{A}}{{\bf{w}}^*}} \right) = 0 \end{array} $$ 因为${ {\bf{w} }^T}$为任意向量，所以： \] {{\bf{A}}^T}{\bf{y}} - {{\bf{A}}^T}{\bf{A}}{{\bf{w}}^*} = 0 $$ 最终可求解最佳参数\({ {\bf{w} }^*} = \left[ { {w_1},{w_2} } \right]\)得： \[ \begin{array}{l} {{\bf{w}}^*} = {{\bf{A}}^{ - 1}}{\left( {{{\bf{A}}^T}} \right)^{ - 1}}{{\bf{A}}^T}{\bf{y}}\\ {{\bf{w}}^*} = {\left( {{{\bf{A}}^T}{\bf{A}}} \right)^{ - 1}}{{\bf{A}}^T}{\bf{y}} \end{array} \label{eq:3} \tag{3} $$ 其中： \] {} = ,{} = $$ 都是可由观测值获得，由于\({\bf{A}}\)为非方阵，所以需要在式\(\eqref{eq:3}\)中求逆时求的是其广义逆，即Moore-Penrose Matrix Inverse。

其中\({ { {\bf{A} }^T}{\bf{A} } }\)的条件数会比较大，表明矩阵病态。

若拟合方程为\(y = {w_1}{x_1} + {w_2}{x_2} + {w_3}{x_3} + \cdots {w_n}{x_n} + {w_0}\)，且给定\(m\)组观测数据，则： $$ {\bf{A}} = {\left[ {\begin{array}{*{20}{c}} {x_1^{\left( 1 \right)}}&{x_2^{\left( 1 \right)}}& \cdots &{x_n^{\left( 1 \right)}}&1\\ {x_1^{\left( 2 \right)}}&{x_2^{\left( 2 \right)}}& \cdots &{x_n^{\left( 2 \right)}}&1\\ {x_1^{\left( 3 \right)}}&{x_2^{\left( 3 \right)}}& \cdots &{x_n^{\left( 3 \right)}}&1\\ \vdots & \vdots &{}& \vdots & \vdots \\ {x_1^{\left( m \right)}}&{x_2^{\left( m \right)}}& \cdots &{x_n^{\left( m \right)}}&1 \end{array}} \right]_{m \times \left( {n + 1} \right)}},{\bf{y}} = {\left[ {\begin{array}{*{20}{c}} {{y^{\left( 1 \right)}}}\\ {{y^{\left( 2 \right)}}}\\ {{y^{\left( 3 \right)}}}\\ \vdots \\ {{y^{\left( m \right)}}} \end{array}} \right]_{m \times 1}} $$ 通过式\(\eqref{eq:3}\)可求得最佳拟合参数。
